# 🎬 Streaming Site - Gestion de Vidéos

📁 Projet FullStack : React.js + Node.js + MongoDB  
💡 Fonctionnalités : Authentification, gestion des vidéos, filtre par utilisateur

---

## 🇪🇸 Español

### 🎯 Descripción
Este proyecto es una plataforma web tipo "Netflix" que permite a los usuarios:
- Registrarse e iniciar sesión con autenticación segura (JWT)
- Agregar videos con título, descripción y URL
- Ver solo sus propios videos
- Eliminar sus videos
- Cambiar la contraseña o eliminar la cuenta

### ⚙️ Tecnologías
- **Frontend:** React.js
- **Backend:** Node.js + Express
- **Base de datos:** MongoDB
